Overview

QATestMate is a productivity extension designed for software testers. It records manual testing activities, captures screenshots, generates structured test steps and exports professional HTML and ZIP reports.

The extension is built with privacy as a priority. All recording is initiated by the user and remains under the user's control.

Information Collected

During an active recording session the extension may process:

  • Mouse clicks
  • Keyboard input required for testing documentation
  • Selected page elements
  • Current page URL
  • Captured screenshots
  • User-entered test information

This information is collected only while the user has started a recording session.

Local Storage

Recorded information, screenshots and extension settings are stored locally inside the browser until the user exports or deletes them.

The developer does not receive or store recorded data.

AI Features

If the user chooses to enable optional AI functionality using their own Google Gemini API Key, requests are sent directly from the user's browser to Google's API.

The developer:

  • does not receive the API Key
  • does not store prompts
  • does not store AI responses
  • does not monitor AI usage

Downloads

Reports are downloaded only when initiated by the user using Chrome's Downloads API.

Data Sharing

QATestMate does NOT:

  • Sell user data
  • Share user data with advertisers
  • Use analytics or tracking services
  • Track browsing history
  • Collect information when recording is inactive
  • Transfer personal information to third parties

Permissions

The extension requests only the permissions required to record browser interactions, capture screenshots, generate reports and export testing evidence.

Security

Users control when recording starts and stops. No hidden monitoring occurs.

Changes

This Privacy Policy may be updated when new functionality is added.
