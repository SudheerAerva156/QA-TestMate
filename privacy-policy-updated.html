QATestMate Privacy Policy

Last Updated: June 29, 2026

Overview

QATestMate is a Chrome extension designed to help Quality Assurance (QA) engineers record software testing activities and generate test evidence reports. The extension only records interactions after the user explicitly starts a recording session. It never records activity automatically.

Information Collected

While a recording session is active, QATestMate may temporarily collect the following:

  • Mouse clicks, text entered into input fields, dropdown selections, and checkbox/radio button selections.
  • Navigation events, element labels, and XPath/CSS selectors of interacted elements.
  • Page titles, Page URLs, and timestamps of each recorded action.
  • Screenshots of the visible browser tab after each recorded step.

Sensitive Information

  • Password fields are automatically detected and their values are never stored.
  • Password values are masked before recording.
  • The extension is not designed to collect payment information or authentication tokens.

How Information Is Used

The recorded information is used solely to create software test documentation, generate HTML reports, generate CSV/JSON exports, and create ZIP packages containing reports and screenshots. No recorded information is used for advertising, analytics, profiling, marketing, or personalization.

Chrome Web Store Limited Use Disclosure

QATestMate strictly adheres to the Chrome Web Store Developer Program Policies. We certify that:

  • We do not sell or transfer user data to third parties, outside of the approved use cases.
  • We do not use or transfer user data for purposes that are unrelated to the item's single purpose of generating QA test documentation.
  • We do not use or transfer user data to determine creditworthiness or for lending purposes.

Data Storage and Transmission

All recorded information is processed and stored locally within the user's browser during the active recording session. Temporary recording data is stored only until the user exports or clears the session. Exported reports are saved directly to the user's computer using the Chrome Downloads API. QATestMate does not upload, transmit, synchronize, or back up any recorded information to external servers. There is no cloud storage, no remote APIs, and no third-party tracking.

User Control

Users have complete control over recordings. You may start recording manually, stop at any time, delete individual steps, clear an entire session, and export reports only when you choose.

Chrome Permissions

QATestMate requests only the permissions required to provide its core functionality. Permissions are used exclusively for recording user interactions, capturing screenshots of the active tab, generating downloadable reports, and temporarily storing recording data to prevent data loss.

Children's Privacy

QATestMate is intended for software testing professionals and is not directed toward children under 13 years of age.

Changes to This Policy

This Privacy Policy may be updated periodically. Any changes will be reflected by updating the "Last Updated" date on this page.
