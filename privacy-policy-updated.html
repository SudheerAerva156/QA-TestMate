QATestMate Privacy Policy

Last Updated: June 25, 2026

Overview

QATestMate is a Chrome extension designed to help Quality Assurance (QA) engineers record software testing activities and generate test evidence reports. The extension only records interactions after the user explicitly starts a recording session. It never records activity automatically.

Information Collected

While a recording session is active, QATestMate may temporarily collect:

  • Mouse clicks
  • Text entered into input fields
  • Dropdown selections
  • Checkbox and radio button selections
  • Navigation events
  • Element labels
  • XPath and CSS selectors of interacted elements
  • Page title
  • Page URL
  • Timestamp of each recorded action
  • Screenshot of the visible browser tab after each recorded step

Sensitive Information

  • Password fields are automatically detected.
  • Password values are never stored.
  • Password values are masked before recording.
  • The extension is not designed to collect payment information or authentication tokens.

How Information Is Used

The recorded information is used solely to:

  • Create software test documentation
  • Generate HTML reports
  • Generate CSV and JSON exports
  • Create ZIP packages containing reports and screenshots
  • Allow users to review and export recorded testing sessions

No recorded information is used for advertising, analytics, profiling, marketing, or personalization.

Data Storage

All recorded information is processed and stored locally within the user's browser during the active recording session.

Temporary recording data is stored only until the user exports or clears the session. Exported reports are saved directly to the user's computer using the Chrome Downloads API.

Data Transmission

QATestMate does not upload, transmit, synchronize, or back up any recorded information to external servers.

  • No cloud storage
  • No analytics
  • No remote APIs
  • No third-party tracking

Third-Party Sharing

QATestMate does not:

  • Sell user data
  • Share user data with third parties
  • Use advertising SDKs
  • Transfer recorded information for marketing purposes

User Control

Users have complete control over recordings and may:

  • Start recording manually
  • Stop recording at any time
  • Delete individual steps
  • Clear an entire session
  • Export reports only when they choose

Chrome Permissions

QATestMate requests only the permissions required to provide its functionality. Permissions are used exclusively for recording user interactions, capturing screenshots of the active tab, generating downloadable reports, and temporarily storing recording data.

Children's Privacy

QATestMate is intended for software testing professionals and is not directed toward children under 13 years of age.

Changes to This Policy

This Privacy Policy may be updated periodically. Any changes will be reflected by updating the "Last Updated" date on this page.
